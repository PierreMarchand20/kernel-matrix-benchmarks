Info

Kernel-Matrix-Benchmarks is a benchmarking environment for operations on kernel matrices. This website contains the current benchmarking results. Please visit http://github.com/kernel-matrix-benchmarks/kernel-matrix-benchmarks/ to get an overview over evaluated data sets and algorithms. Make a pull request on Github to add your own code or improvements to the benchmarking system.

Results

Results are split by kernel and dataset. At the bottom of the page, you can find an overview of an algorithm's performance on all datasets. The plots shown depict Precision (such as an L1, L2 or L-infinity error, depending on the context) against Run time. Clicking on a plot reveals detailed interactive plots, including approximate precision and pre-computation time.
